# HopeForward NGO Website

A simple NGO website developed using HTML, CSS, and JavaScript.

This project was created as a university group project. The website is designed with a beginner-friendly approach while covering the important sections commonly found on NGO websites.

## Project Objective

The purpose of this website is to present an NGO's mission, activities, donation system, and contact information through a simple and user-friendly interface.

## Features

- Responsive Navigation Bar
- Hero / Welcome Section
- About Us Section
- Previous Work & Projects
- Upcoming Programs
- Team Section
- Donation Section
- Contact Form
- Admin Panel (password:hopeforward2026)
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript
